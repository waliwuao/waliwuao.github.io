小白也能懂的 Conda 入门指南

发表于 2026-03-06 分类于 小白都能懂

前言

Conda 是一款开源的包管理与环境管理工具。它不仅能轻松安装、更新、卸载 Python 库，更能创建相互隔离的虚拟环境，确保不同项目使用不同版本的依赖，从而避免版本冲突。无论是数据分析、机器学习还是日常开发，Conda 都是高效管理开发环境的利器。

对于初学者，建议安装更轻量的 Miniconda，它已完全覆盖日常开发的核心需求。

一、 Conda 的安装

在正式安装前，请明确你的开发需求：

  • VS Code 用户：Conda 是 Python 开发的必备工具。
  • PyCharm 用户：PyCharm 内置了 venv，若偏好使用 Conda，可在创建项目时选择 Conda 环境，软件会自动指引完成安装。

安装步骤

  1. 下载安装包：访问 Anaconda 官网 进行下载（需注册账号，建议使用 GitHub 快速登录）。
  2. 初始化配置：安装过程中，若提示「是否在终端默认初始化」，请务必选择 Yes。
  3. 环境变量：确保安装程序已将 Miniconda 的路径添加到系统环境变量中。

验证安装

打开终端（Windows 可按 Win + R 输入 cmd），若命令行路径前出现 (base) 标识，即表示安装成功。若未显示，可尝试手动运行 conda activate 激活。

提示：Windows 系统拥有多种终端（如 PowerShell, CMD）。若在 VS Code 的集成终端中未显示 (base)，请点击终端面板右上角的下拉箭头，切换不同的终端类型尝试。

四、 特别提示

  • 关于 pip 的使用：在激活 Conda 环境的状态下直接使用 pip，安装的包会自动归入当前 Conda 环境。此外，Conda 环境已内置 pip，无需额外安装。
  • 终端生效范围：激活的 Conda 环境仅在当前终端窗口有效。若打开新的终端，需重新执行 conda activate。
  • VS Code 的运行方式：VS Code 的「Code Runner」插件默认使用系统全局 Python。若需在特定 Conda 环境下运行代码，请确保在终端中先执行 conda activate，随后在该终端内运行 python <文件名.py>。
